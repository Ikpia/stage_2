## Run Locally

Clone the project

```bash
  git clone https://github.com/Ikpia/stage_2
```

Go to the project directory

```bash
  cd stage_2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
    npm start

```

Start mongodb

```bash
    mongod

```

Test the endpoints

```bash
    Make sure to have postman installed then use it to test the endpoints

```
